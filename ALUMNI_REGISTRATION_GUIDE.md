# Alumni Registration - Quick Reference

## Two Ways to Add Alumni

### Method 1: Promote Existing Student
**When to use**: Student already exists in the system and is graduating

**Required Information**:
- Student Internal ID
- Graduation Year

**Process**:
1. Click "+ Create Alumni"
2. Select "Promote Existing Student"
3. Enter Student ID and Graduation Year
4. Click "Promote to Alumni"

**Result**: Student's status changes to 'alumni', appears in Alumni Directory

---

### Method 2: Register New Alumni (NEW!)
**When to use**: Alumni who graduated before the system existed

**Required Information**:
- First Name *
- Last Name *
- Graduation Year *

**Optional Information You Can Provide**:
- Personal Details (Email, DOB, Gender, Address, etc.)
- Academic Info (Class graduated from)
- Professional Info (Current job, company, university, skills, bio)
- Social Links (LinkedIn, Twitter, Portfolio)
- Parent/Guardian Info (for records)

**Process**:
1. Click "+ Create Alumni"
2. Select "Register New Alumni"
3. Fill in the comprehensive form
4. Click "Register Alumni"

**Result**: 
- New alumnirecord created
- Credentials generated and displayed
- Alumni can access Alumni Portal immediately

## Common Use Cases

### Scenario 1: School just adopted the system
- Use **Direct Registration** to add alumni from previous years (2015, 2016, 2017, etc.)
- Use **Promotion** for current students graduating this year

### Scenario 2: Alumni reunion planning
- Use **Direct Registration** to add alumni who attended the reunion
- Capture their current professional information
- They can immediately access the Alumni Portal

### Scenario 3: Regular graduation
- Use **Promotion** for all graduating students
- Student data transfers automatically
- Minimal data entry required
